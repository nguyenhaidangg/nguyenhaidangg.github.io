Fashion Shop (Hybrid & Distributed DB)

10/2025 -- 12/2025

Mô tả: Hệ thống Thương mại điện tử phân tán vùng miền. Nền tảng E-commerce áp dụng mô hình Hybrid Database, tự động định tuyến và hiển thị sản phẩm dựa trên khu vực địa lý của người dùng.

Công nghệ: Vue 3, Node.js, TypeScript, SQL Server (Replication), MongoDB, Cloudinary, VNPay.

Vai trò & Đóng góp:

  • Kiến trúc Phân tán (Distributed System): Cấu hình SQL Server Replication (Distributor, Publication, Subscription) để phân mảnh và đồng bộ dữ liệu giao dịch từ Server Trung tâm về 3 chi nhánh (Hà Nội, Đà Nẵng, TP.HCM).
  • Hybrid Database: Triển khai mô hình lưu trữ đa hệ quản trị. Sử dụng SQL Server cho các giao dịch lõi (đảm bảo ACID) và MongoDB để lưu trữ dữ liệu phi cấu trúc, tích hợp Cloudinary quản lý Media.

Kết quả: Xây dựng thành công cơ sở dữ liệu phân tán, đảm bảo tính nhất quán (Data Consistency) trên toàn bộ các node và xử lý mượt mà nghiệp vụ thanh toán, giỏ hàng theo từng chi nhánh.
